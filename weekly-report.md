# Self-Improving 周报生成器

## 职责
每周日向进化群（oc_8e02a9ced0671cac8413b4c98e76637a）发送 self-improving 进展报告，在进化周报（20:00）之前发送。

## 报告内容
1. **corrections.md** — 本周新增的纠正条目
2. **domains/** — 各领域经验文件的更新情况
3. **待改进点** — 本周发现的执行问题
4. **下周改进计划**

## 发送时间
周日 19:30（进化周报前30分钟）

## 格式
```
🦞 Self-Improving 周报 | {date}

📝 corrections.md 本周新增：
- （条目列表）

📚 domains/ 领域经验更新：
- skill-management.md：（新增内容摘要）
- cron-management.md：（新增内容摘要）
- memory-system.md：（新增内容摘要）
- content-workflow.md：（新增内容摘要）

⚠️ 本周待改进：
- （问题描述）

🎯 下周计划：
- （改进方向）
```
